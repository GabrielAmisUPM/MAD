## Description
Weather map app \
Add point on the map that are saved, real time weather on your location and the marker you added.

## Feature

- Fireship login
- Map
- Persitence data (login)
- CSV Save file for coordinates point
- Navigation menu system
- Toat Message/Text
- API Weather
